# Thư mục hình ảnh

Thư mục này chứa các hình ảnh được sử dụng trong website.

## Cấu trúc

```
images/
├── before-after/     # Hình ảnh trước/sau điều trị
├── services/         # Hình ảnh dịch vụ
├── team/             # Hình ảnh đội ngũ bác sĩ
└── gallery/          # Thư viện hình ảnh khác
```

## Hướng dẫn tối ưu hình ảnh

1. Định dạng:
   - JPG cho hình ảnh có nhiều màu sắc
   - PNG cho hình ảnh cần trong suốt
   - WebP cho cả hai trường hợp trên (khuyến nghị)

2. Kích thước:
   - Ảnh hero: tối đa 1920x1080px
   - Ảnh dịch vụ: 800x600px
   - Ảnh thumbnail: 400x300px

3. Tối ưu:
   - Sử dụng công cụ nén ảnh
   - Thêm alt text đầy đủ
   - Đặt tên file có ý nghĩa
